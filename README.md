# Supermarket Sales Analysis

## Intern Details
Intern ID: CITS1448

## Project Overview
This project focuses on analyzing supermarket sales data using Python. The objective is to identify sales patterns, customer purchasing behavior, and business insights through data visualization and exploratory data analysis (EDA).

## Objectives
- Analyze sales performance.
- Identify top-performing product categories.
- Study customer purchasing behavior.
- Understand payment preferences.
- Generate business insights using visualizations.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## Dataset
The dataset contains supermarket sales records including product lines, customer types, payment methods, ratings, and sales information.

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Customer Behavior Analysis
- Visualization of Key Metrics

## Key Insights
- Identified top-selling product categories.
- Analyzed customer purchase patterns.
- Examined payment method preferences.
- Evaluated branch-wise sales performance.

## Conclusion
The project provided valuable insights into supermarket operations and customer behavior, helping understand factors influencing sales performance.
